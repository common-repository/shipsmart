== Installation ==

1. Faça upload deste plugin em seu WordPress, e ative-o;
2. Entre no menu lateral “WooCommerce > ShipSmart”;
3. Ative a API KEY da ShipSmart.
4. Após ativar a API KEY, configurar endereço da loja  “WooCommerce > Configurações > Geral > Endereço da loja”;
5. Configurar as áreas de entrega “WooCommerce > Configurações > Entrega > Áreas de entrega” e adicionar o método de entrega “ShipSmart”;
6. Habilitar a opção de entrega “ShipSmart”;
7. Editar as configurações de “Dados da loja”, “Taxas”, “Prazo adicional”; “Renomear rótulos”
8. Salvar alterações.

== Frequently Asked Questions ==

= Dúvidas sobre a ShipSmart? =
Acesso nosso painel de dúvidas [aqui](https://shipsmart.com.br/tire-suas-duvidas/).

= Ao instalar o plugin, devo fazer alguma coisa? =
Instalando o plugin você poderá vender os seus produtos para mais de 140 países. Não será necessário criar nenhum campo customizado, o próprio APP criará um campo de HS code na sessão de produtos e você só precisa preencher corretamente o HS code (6 primeiros dígitos do NCM)

= Como configurar as regras de frete? =
É só ir em configurações > entregas > áreas de entrega > ShipSmart - Lá será possível fazer todas as configurações de subsídio de frete, taxas e definição se terá seguro ou não.

= Como despachar o meu pedido? =
Quando a ordem de venda estiver disponível na sessão "Pedidos" do Woocommerce, será possível clicar nos pedidos e adicionar as informações necessárias para seguir com o embarque. As informações são - chave da nota fiscal (44 dígitos, sem espaço) e caixas (que devem ser criadas na instalação do plugin), com isso já é possível criar o pedido na ShipSmart e ter a etiqueta de embarque e os documentos necessários de embarque dentro do próprio pedido.
